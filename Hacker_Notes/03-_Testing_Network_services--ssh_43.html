SSH Port forwarding

  • generate ssh keys
    • ssh-keygen -t rsa -b 4096 -C 'drt@htb' -f secret.htb -P ''
  • create a folder of the user in victim
    • export PUBLIC_KEY=$(cat secret.htb.pub)
    • mkdir -p /home/dasith/.ssh
    • echo $PUBLIC_KEY >> /home/dasith/.ssh/authorized_keys

ssh o command

  • ssh -o "UserKnownHostsFile=/dev/null" -o "StrictHostKeyChecking=no" student@$target -p2222

Debug errors

  • no matching key exchange method found + o matching host key type found. Their offer: ssh-rsa,ssh-dss
    • ssh -oKexAlgorithms=+diffie-hellman-group1-sha1 -oHostKeyAlgorithms=+ssh-dss j0hn@10.11.1.252 -p22000
  • Permanent solution
# ~/.ssh/config 
Host 10.11.1.252
	KexAlgorithms +diffie-hellman-group1-sha1
	HostKeyAlgorithms +ssh-dss
