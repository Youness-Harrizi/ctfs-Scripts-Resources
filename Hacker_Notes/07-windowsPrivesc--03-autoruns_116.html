03-autoruns


Vulnerability :

  • autorun program has FILE_ALL_ACCESS permissions

  • AutoRun executables is writable by everyone

 

Exploit :

  • enumerate:
    • reg query HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run // query the registry for the autorun executables

  • replace the autoruns  executable with a reverse shell  

  • wait until the administrator connects to the machine
